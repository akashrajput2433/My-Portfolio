# Akash Singh Portfolio

A premium personal portfolio website built to showcase my professional journey, AI/ML-oriented projects, software engineering experience, technical skills, certifications, and achievements in a modern, recruiter-focused format.

This portfolio is designed with a strong emphasis on premium UI, smooth interactions, responsive design, and professional storytelling to create a strong first impression for recruiters, hiring managers, and collaborators.

## ⚡ Quick Start

### Prerequisites

- Node.js 18 or higher
- npm installed

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/akashrajput2433/My-Portfolio.git
cd My-Portfolio
```

2. **Install dependencies**
```bash
npm install
```

3. **Run the development server**
```bash
npm run dev
```

4. **Open in browser**

Navigate to:

```bash
http://localhost:5173/
```

## 📘 Usage Guide

### Portfolio Content Customization

You can update the portfolio content from:

```bash
src/data/portfolioData.js
```

This file controls:
- Hero content
- About section
- Skills
- Projects
- Experience
- Education
- Certifications
- Achievements
- Contact details

### UI Customization

Main design and theme styling can be adjusted from:

```bash
src/index.css
```

Reusable UI components live in:

```bash
src/components/
```

Portfolio sections live in:

```bash
src/sections/
```

## 🌐 Deployment

### Deploy on Vercel

1. Push the project to GitHub
2. Import the repository into Vercel
3. Vercel will auto-detect Vite settings
4. Deploy and get your live URL

### Deploy on Netlify

1. Push the project to GitHub
2. Import the repository into Netlify
3. Set build command:
```bash
npm run build
```
4. Set publish directory:
```bash
dist
```
5. Deploy the site
